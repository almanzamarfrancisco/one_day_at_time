One day at a time

What is this?

This is a simple web application that helps you to keep track of your daily tasks. You can add, edit and delete tasks. You can also mark tasks as completed.

How to use

Click on the "Add task" button to add a new task. Click on the "Edit" button to edit a task. Click on the "Delete" button to delete a task. Click on the checkbox to mark a task as completed.

Features

  • Add tasks
  • Edit tasks
  • Delete tasks
  • Mark tasks as completed

Technologies used

  • HTML
  • CSS
  • JavaScript
